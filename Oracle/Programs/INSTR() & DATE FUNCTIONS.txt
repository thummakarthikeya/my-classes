                            CHARACTER FUNCTIONS

1.Check following instr() queries outputs


instr():-it is used to check whether the substring is existed 
=======         in the given string or not.


•if sub string is existed in the given string, it returns the position.

•if sub string is not existed in the given string ,it returns 0.


syntax: instr(columnname(or)string , sub string, searching position,
======            No of occurrences from searching position);


Searching position -> +ve or –ve
Searching and No of occurrences -> must be numbers.


         always instr() returns position based on last 2 parameters. 
but oracle server counts number of characters always left side first 
position onwards.



SQL> select instr('ABCDEFGHCDKMNBCDJ','CD',1,1) from dual;



SQL> select instr('ABCDEFGHCDKMNBCDJ','CD',5,2) from dual;



SQL> select instr('ABCDEFGHCDKMNBCDJ','CD',-5,2) from dual;



SQL> select instr('krishna stami lord krishna jai krishna','krishna',1,2) from dual;



SQL> select instr('krishna stami lord krishna jai krishna','krishna',-1,3) from dual;



SQL> select instr('gowtham raj','raj') from dual;



SQL> select instr('gowtham raj','sai') from dual;



SQL> select instr('ABC*D','*') from dual; // it returns the position of the 
                                             delemiters(special characters)
